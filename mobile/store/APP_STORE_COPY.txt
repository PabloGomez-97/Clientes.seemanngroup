ENGLISH (recommended for App Review):

Thank you for reviewing Seemann Group.

WHAT THIS APP IS
This is a B2B client portal for existing Seemann Group logistics customers. There is NO public sign-up inside the app. User accounts are created by Seemann Group for contracted clients only. Login uses the same credentials as the web portal: https://portalclientes.seemanngroup.com

DEMO ACCOUNT (required to access all features)
Username / Email: EMAIL_PRUEBA
Password: PASSWORD_PRUEBA

If a Cloudflare Turnstile captcha appears after failed attempts or on first login, please complete the captcha widget and try again.

HOW TO TEST
1. Open the app → enter the demo email → Continue → enter the password → Log in.
2. Home (Inicio): welcome header, shipment/document counters, shortcuts.
3. Bottom tabs: Seguimientos (tracking), Operaciones (air/ocean/ground shipments), Cotizaciones (quotes), Más (more).
4. Operaciones: browse shipment lists and open a detail if available for this account.
5. Cotizaciones: browse quotes and open a detail if available.
6. Seguimientos: view tracking list / map embed when the account has trackings.
7. Más → Notificaciones: toggle push notification preference (optional; may request system permission).
8. Más → Política de Privacidad / Términos de Servicio: in-app legal documents.
9. Más → Eliminar cuenta: opens a mailto request flow to privacidad@seemanngroup.com (account deletion is processed manually within ~30 business days; no public self-serve signup).

NOTES
- Some modules may show empty states if the demo account has no data in that module; that is expected.
- The app requires network access to portalclientes.seemanngroup.com and related APIs.
- Push notifications are optional and can be disabled in Más → Notificaciones.
- Privacy Policy (public URL): https://www.seemanngroup.com/privacidad
- Support: https://www.seemanngroup.com/contacto
- Encryption: standard HTTPS only (no non-exempt encryption). ITSAppUsesNonExemptEncryption = false.
